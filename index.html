웹 Web이란?

WWW의 정의

  • World Wide Web의 줄임말
  • 전 세계적으로 분산된 수많은 문서와 기타 리소스들이 서로 연결되어 있는 거대 정보의 네트워크
  • 일반적으로 웹페이지라고 불리는 문서 형식으로 제공
  • HTML이라는 마크업 언어 작성

웹의 동작 원리

    요청
  • 사용자가 웹 브라우저 주소창에 URL을 입력하면 브라우적 해당 웹 서버에 페이지 정보 요청
    응답
  • 서버는 요청받은 웹 페이지의 데이터를 조회하고 처리하여 브라우저에게 전달
    렌더링
  • 전달받은 리소스를 조합하여 사용자가 볼 수 있는 화면으로 보여줌

프론트엔드와 백엔드

    프론트 엔드
  • 사용자 인터페이스를 다루고, 웹 애플리케이션과 상호작용하고 정보를 볼 때 필요한 모든 것 구현
  • HTML, CSS, Javascript 등 사용
    백엔드
  • 서버와 데이터 처리를 담당하고 데이터 저장,관리 및 서버 로직 구현
  • Python, Java, Node.js, SQL 등 사용

HTML,CSS,Javescript 개념

    HTML
  • HyperText Markup Language
  • 웹페이지의 구조를 만드는 뼈대 역할
  • HyperText : 정해진 순서 없이 참조를 통해 한 문서에서 다른 문서로 접근할 수 있는 텍스트
  • Markup Language : 콘텐츠의 구조를 정의하고 문서의 구조와 의미를 태그로 감싸서 표현
    CSS
  • Casading Style Sheets
  • 웹페이지의 모양과 표현을 담당하는 스타일 시트 언어
  • HTML로 만든 구조를 예쁘게 꾸며주는 역할
    Javscript
  • 웹페이지의 기능과 동작을 담당
  • 사용자와 상호작용하거나 동적인 기능을 추가하기 위해 필요
